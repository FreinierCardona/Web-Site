1. Etiquetas Estructurales

<header>

Explicación: Define la cabecera de una página o sección. Generalmente contiene logo, título, menú de navegación o introducción..

Uso:Se utiliza para mostrar la parte superior del sitio web.

<header>
    <h1>Sena</h1>
    <p>Sofia Plus</p>
</header>

Sena

Sofia Plus

<nav>

Explicación: Contiene enlaces de navegación para el sitio web.

Uso: Se utiliza para crear un menú de navegación que permita al usuario moverse por el sitio.

<nav>
    <a href="#inicio">Inicio</a>
    <a href="#acercaDe">Acerca de</a>
    <a href="#contacto">Contacto</a>
</nav>
Inicio Acerca De Contacto

<section>

Explicación: Representa una sección temática del contenido.

Uso:Sirve para dividir el contenido en partes organizadas.

<section>
    <h3>Example</h3>
    <p>Lorem ipsum, dolor sit amet consectetur adipisicing elit. </p>
</section>

Example

Lorem ipsum dolor sit amet consectetur adipisicing elit.

<footer>

Explicación: Define el pie de página de una página o sección. Generalmente contiene información de copyright, enlaces a políticas de privacidad o contactos.

Uso: Se utiliza para mostrar información adicional al final del sitio web.

<footer>
    <p> 2026 Mi Web. Todos los derechos reservados.</p>
</footer>

2026 Mi Web. Todos los derechos reservados.

2. Etiquetas de Texto

<h1> al <h6>

Explicación: Definen los encabezados del documento.

Uso: Estructuran jerárquicamente la información (el h1 es el más importante).

<h1>Título Principal</h1>
<h2>Subtítulo</h2>

Título Principal

Subtítulo

<p>

Explicación: Definen los párrafos de texto en el documento.

Uso: Se utilizan para mostrar el contenido principal del sitio web.

<p>Este es un párrafo de texto.</p>

Este es un párrafo de texto.

<strong>

Explicación: Indica que el texto tiene gran importancia.

Uso: Se utiliza para resaltar palabras o frases importantes dentro de un párrafo.

<strong>Este es un texto en negrita.</strong>

Este es un texto en negrita.

<i>

Explicación:mostrar texto en cursiva (italic).

Uso: Se utiliza normalmente para palabras extranjeras, términos técnicos, títulos o frases que se quieren destacar visualmente.

<i>Este es un texto en cursiva.</i>
Este es un texto en cursiva.

3. Etiquetas Multimedia

<img>

Explicación: Inserta una imagen en el documento.

Uso: Requiere el atributo src para la ruta y alt para el texto alternativo.

<img src="library/assets/images/Adso.png" alt="Sena">

<video>

Explicación: Permite reproducir videos directamente en la página.

Uso:Se utiliza para contenido audiovisual.

<video src="library/assets/videos/Millonario.mp4" controls></video>

<audio>

Explicación: Permite reproducir audio directamente en la página.

Uso: Se utiliza para contenido de audio.

<audio controls>
    <source src="library/assets/audio/Music.mp3" type="audio/mpeg">
</audio>

<iframe>

Explicación: Permite insertar contenido de otras páginas web.

Uso: Se utiliza para contenido de audio.

<iframe src="https://betowa.sena.edu.co/" controls></iframe>

4. Formularios

<input>

Explicación: Campo de entrada de datos.

Uso: Permite al usuario escribir texto, contraseñas, seleccionar fechas, etc.

<input type="text" placeholder="Escribe un mensaje">

<form>

Explicación: Contenedor para agrupar elementos de formulario.

Uso: Permite organizar y estructurar los elementos de entrada del usuario.

<form>
    <label for="nombre" class="form-label">Nombre<label>
    <input type="text" class="form-control" id="nombre" placeholder="Escribe tu nombre">
</form>

<label>

Explicación: Etiqueta que describe el propósito de un campo de entrada.

Uso: Mejora la accesibilidad y la experiencia del usuario al proporcionar contexto para los campos de formulario.

<label for = "correo"> Correo</label>
<input type="email" id="correo" placeholder= "Escribe tu correo">

<button>

Explicación: Elemento que actúa como un botón para enviar un formulario o realizar una acción.

Uso: Se utiliza para permitir que los usuarios interactúen con el formulario de manera intuitiva.

<button type="submit">Enviar</button>

5. Tablas

<table>

Explicación: Define una tabla de datos.

Uso: Es el contenedor principal donde irán las filas y columnas. 

<table>
    <tr>
        <th>Nombre</th>
        <th>Edad</th>
    </tr>

    <tr>
        <td>Juan</td>
        <td>20</td>
    </tr>

    <tr>
        <td>Ana</td>
        <td>22</td>
    </tr>

</table>
Nombre Edad
Carlos 20
Ana 22

<tr>

Explicación: Define una fila en una tabla.

Uso: Se utiliza para crear filas dentro de una tabla.

<table>
    <tr>
        <td>Celda 1</td>
        <td>Celda 2</td>
    </tr>
</table>
Celda 1 Celda 2

<th>

Explicación: Define una celda de encabezado en una tabla.

Uso: Se utiliza para crear celdas de encabezado dentro de una fila de tabla.

<table>
    <tr>
        <th>Nombre</th>
        <th>Edad</th>
    </tr>
</table>
Nombre Edad

<td>

Explicación: Define una celda de datos en una tabla.

Uso: Se utiliza para crear celdas de datos dentro de una fila de tabla.

<table>
    <tr>
        <td>Juan</td>
        <td>20</td>
    </tr>
</table>
Juan 20
