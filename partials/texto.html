Uso: Se utilizan para mostrar el contenido principal del sitio web.

<strong>

Explicación: Indica que el texto tiene gran importancia.

Uso: Se utiliza para resaltar palabras o frases importantes dentro de un párrafo.

<strong>Este es un texto en negrita.</strong>

Este es un texto en negrita.

<i>

Explicación:mostrar texto en cursiva (italic).

Uso: Se utiliza normalmente para palabras extranjeras, términos técnicos, títulos o frases que se quieren destacar visualmente.

<i>Este es un texto en cursiva.</i>
Este es un texto en cursiva.
