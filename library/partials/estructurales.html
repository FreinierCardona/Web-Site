1. Etiquetas Estructurales

<header>

Explicación: Define la cabecera de una página o sección. Generalmente contiene logo, título, menú de navegación o introducción..

Uso:Se utiliza para mostrar la parte superior del sitio web.

<header>
    <h1>Sena</h1>
    <p>Sofia Plus</p>
</header>

Sena

Sofia Plus

<nav>

Explicación: Contiene enlaces de navegación para el sitio web.

Uso: Se utiliza para crear un menú de navegación que permita al usuario moverse por el sitio.

<nav>
    <a href="#inicio">Inicio</a>
    <a href="#acercaDe">Acerca de</a>
    <a href="#contacto">Contacto</a>
</nav>
Inicio Acerca De Contacto

<section>

Explicación: Representa una sección temática del contenido.

Uso:Sirve para dividir el contenido en partes organizadas.

<section>
    <h3>Example</h3>
    <p>Lorem ipsum, dolor sit amet consectetur adipisicing elit. </p>
</section>

Example

Lorem ipsum dolor sit amet consectetur adipisicing elit.

<footer>

Explicación: Define el pie de página de una página o sección. Generalmente contiene información de copyright, enlaces a políticas de privacidad o contactos.

Uso: Se utiliza para mostrar información adicional al final del sitio web.

<footer>
    <p> 2026 Mi Web. Todos los derechos reservados.</p>
</footer>
